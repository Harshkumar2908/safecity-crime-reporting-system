# 🛡️ SafeCity – Crime Reporting System

A full-stack web application that allows users to report crimes, track reports, and manage cases efficiently.  
Built using Java (JSP/Servlet), MySQL, and deployed on Apache Tomcat.

---

## 🌐 Live Demo

👉 https://safecity-crime-reporting-system.onrender.com/SafeCity/

---

## 📌 Features

- 👤 User Registration & Login
- 📝 Report Crime Online
- 📊 Admin Dashboard
- 📂 View & Manage Reports
- 🔄 Update Report Status
- 🔐 Session Management & Authentication
- 💾 Database Integration (MySQL)

---

## 🛠️ Tech Stack

**Frontend:**
- HTML5
- CSS3
- Bootstrap

**Backend:**
- Java (JSP & Servlets)
- JDBC

**Database:**
- MySQL

**Server:**
- Apache Tomcat

**Deployment:**
- Render (Docker-based deployment)
